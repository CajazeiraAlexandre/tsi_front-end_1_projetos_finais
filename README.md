# 💻 Projetos Finais - Front-End 1 (TSI)

Bem-vindo(a) ao repositório oficial de projetos finais da disciplina de **Desenvolvimento Web Front-End I** do curso de **Tecnologia em Sistemas para Internet (TSI)** do Instituto Federal do Piauí (IFPI) - Campus São Raimundo Nonato.

Este espaço é dedicado a registrar, organizar e dar visibilidade aos trabalhos práticos desenvolvidos pelos alunos ao longo dos semestres. Ele funciona como um **portfólio público** para as turmas e também como um ambiente real de prática de versionamento colaborativo utilizando Git e GitHub.

## 🎯 Sobre a Disciplina

O objetivo principal da disciplina é capacitar os estudantes na construção de interfaces web modernas, acessíveis e responsivas. Ao longo do curso, o foco prático envolve:

* **Fundamentos da Web:** Entendimento sobre protocolos, relação cliente-servidor e funcionamento dos navegadores.
* **Estruturação (HTML5):** Uso correto de semântica, formulários, tabelas, multimídia e aplicação de diretrizes de acessibilidade.
* **Estilização (CSS3):** Criação de layouts com Flexbox e Grid Layout, além da adoção de abordagens *mobile-first* e *desktop-first*.
* **Interatividade Básica:** Introdução ao JavaScript para manipulação da DOM e eventos.
* **Práticas de Mercado:** Versionamento de código em equipe (Git/GitHub) e publicação de projetos (GitHub Pages).
